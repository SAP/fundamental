Identifier is a way to visually present something using an icon or user initials.
{: .docs-intro}

There are multiple sizes:
- Default matches the base font size (14px)
- Extra Extra Small (`--xxs` - 20px)
- Extra Small (`--xs` - 28px)
- Small (`--s`) - 32px
- Medium (`--m`) - 48px
- Large (`--l`) - 64px
- Extra Large (`--xl`) - 88px
- Extra Extra Large (`--xxl`) - 112px

<br>
